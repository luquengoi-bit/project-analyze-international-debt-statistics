# Analysis of international debt

## Objective
To analyse the debt of developing countries using SQL.

## Dataset
World Bank data on international debt.

## Analyses
- Number of countries
- Countries with the highest debt
- Countries with the lowest repayment

## Results
- China has the highest debt
- Timor-Leste has the lowest repayment

## Technologies used
- SQL (queries)
- Jupyter Notebook (.ipynb)

## Files
- analysis.ipynb → full analysis
- queries.sql → SQL queries
